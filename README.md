# LifeOS Site

Public landing page for LifeOS.

This repo exists separately from the operating system repo so the public site can evolve fast while the main LifeOS codebase stays focused on the OS itself.

## Stack

- React 19
- Vite
- Tailwind CSS v4
- Motion

## Local development

```sh
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Production build

```sh
npm run build
npm run preview
```

## Notes

- The site is intentionally honest about the current state of LifeOS.
- External channels like newsletter, YouTube, and Twitch should only be linked once they are actually live.
- SEO metadata, structured data, `robots.txt`, and `sitemap.xml` are maintained in this repo for the public landing.
